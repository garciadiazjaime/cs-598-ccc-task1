BACKGROUND

The data contained in the compressed file has been extracted from the DB1BCoupon data table of the "Origin and Destination Survey" database from the TranStats data library. The time period is indicated in the name of the compressed file; for example, XXX_XXXXX_2001_1 contains data of the first quarter of the year 2001.
